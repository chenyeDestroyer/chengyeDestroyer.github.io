# Rhein Lab - Document System

莱茵生命风格资料管理系统 | Rhine Lab Document System

## Features
- Create, import, and browse materials (PDF / DOCX / PNG)
- Handwriting notes with 7 colors, pen/marker/eraser tools
- Page flip, jump, rotate controls
- Sidebar quick navigation with last-position memory
- 5 color themes + dark mode + font size adjustment
